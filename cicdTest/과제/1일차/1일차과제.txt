01	nginx 이미지를 Docker Hub에서 로컬로 다운로드하시오.                   docker pull nginx
02	로컬에 다운로드된 모든 이미지의 목록을 확인하시오.	                    docker images
03	nginx 이미지를 web-server라는 이름으로 백그라운드(-d)에서 실행하시오.	docker run --name web-server -d nginx
04	현재 실행 중인 컨테이너의 목록을 확인하시오.	                        docker ps
05	종료된 컨테이너를 포함하여 모든 컨테이너 목록을 확인하시오.	             docker ps -a
06	호스트의 8080번 포트를 컨테이너의 80번 포트로 연결하여 nginx를 실행하시오.  	docker run -d -p 8080:80 nginx
07	실행 중인 web-server 컨테이너를 정지시키시오.	                        docker stop web-server
08	정지된 web-server 컨테이너를 삭제하시오.	                            docker rm web-server
09	로컬에 있는 nginx 이미지를 삭제하시오.	                                docker rmi nginx
10. 모든 컨테이너를 중지 시키시요                                           docker stop $(docker ps -aq)
