#### pre_산업일자리_산업밀도.csv

1. 전체산업밀도(천명) = 전체산업사업체수 / (인구 천명당 / 종사자수)
2. 통계방식변경(~ 2019년까지 조사기준, 2020년 이후 등록기준)으로 보정을 위해 통계방식변경 컬럼 생성

#### pre_산업일자리_임금수준.csv

1. 상용월평균급여(원)(지역별 임금수준) = 상용월급여액(원) + 상용특별급여(원)
2. 세종시 : 2017년 충남에서 분리 후 (2019. 10.) 고용부문 ｢17개 시도별 고용통계｣ 공표 및 공표 규모 세분화로 2020년 부터 세종특별자치시를 포함한 17개 행정구역으로 분류.
3. 따라서 2015~2019년 결측 데이터는 2020년 이후 충남 대비 세종의 비율을 구해 추정값 사용.
