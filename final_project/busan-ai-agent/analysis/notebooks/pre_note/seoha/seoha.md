#### 2. 조회설정

* [ ] 연령 및 성별 인구

- 항목: 내국인(명)
- 행정구역별(읍면동): 전국, 서울, 부산, 대구, 인천, 광주, 대전, 울산, 세종, 경기, 강원, 충북, 충남, 전북, 전남, 경북, 경남, 제주 (읍면동만 제외)
- 연령별: 19세, 20-24세, 25-29세, 30-34세, 35-39세
- 시점: 2015년-2024년

- [ ] 주민등록연앙인구

- 행정구역(시군구)별: 전국, 서울, 부산, 대구, 인천, 광주, 대전, 울산, 세종, 경기, 강원, 충북, 충남, 전북, 전남, 경북, 경남, 제주 (전체)
- 연령별: 계
- 시점: 2015년-2023년, 2024년

- [ ] 개념

1. 주민등록연앙인구 = (t년1월1일 인구 + t년12월31일 주민등록 거주자 인구) / 2 (1월 1일 인구는 공표하지 않아 실제 t-1년 12.31일 인구를 사용)
   : 연말기준 주민등록인구를 기초로 연앙(年央)개념으로 재작성하여 제공하는 일반가공 통계
   (주민등록인구는 거주자, 거주불명자, 재외국민이 포함되어 있는데 연앙인구는 그중 거주자로 재작성됨)
   -> 당해 연평균 인구의 개념, 출산·사망·이동 등 각종 인구동태율 산출 시 분모인구로 활용
2. 연앙인구 : 연앙(年央)인 7.1일의 인구 -> 연초(1.1일)와 연말(12.31) 인구의 산술평균 개념을 사용하여 계산된 인구)

* 사용 예시 : 조출생률 = 특정 1년간의 총 출생아수 / 당해연도의 연앙인구 × 1,000
  (인구 1,000명당 출생아수 -> 인구규모가 각기 다른 지역이나 시점 간의 출산 수준을 비교할 때 매우 유용한 지표)

3. 인구총조사는 ‘실제로 사는 사람’, 주민등록연앙인구는 ‘등록된 사람’을 세기 때문에 주민등록연앙인구가 더 크게 나옴

### <1차 전처리>

#### 1. 연령 및 성별 인구

- 멀티 인덱스 제거 (중복된 항목 인덱스('내국인(명)') 제거)
- 컬럼명 변경 : 시점 -> 연도, 행정구역별(읍면동) -> 행정구역

#### 2. 주민등록연앙인구

- 멀티 인덱스 제거 (중복된 항목 인덱스('내국인(명)') 제거)
- 컬럼명 변경 : 시점 -> 연도, 행정구역(시군구)별 -> 행정구역, 계 -> 연앙인구수(명)
- 값 변경 : 두 데이터의 값 통일을 위해 2015~2023 데이터에서 '강원도 -> 강원특별자치도', '전라북도 -> 전북특별자치도'로 변경
- 천명 당 연앙인구수 컬럼('연앙인구수(천명)') 추가
